# LegacySMS
Güçlü bir SMSBOMBER
-Çalışmayan ve Gereksiz SMS'ler silindi.

-Bilgisayar Aracılığıyla artık pyton indirmeden anında kullanılabilir.

-Arayüz ve Bilgilendirme eklendi.

-Tema değişti.

-Hatalar giderildi.
